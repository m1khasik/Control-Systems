# Система управления дефектами строительных объектов

Веб-приложение для контроля дефектов на строительных объектах.

## Стек технологий
- **Бэкенд**: Python Django, PostgreSQL
- **Фронтенд**: React.js, Bootstrap
- **Инфраструктура**: Docker, Nginx

# Локальный запуск
